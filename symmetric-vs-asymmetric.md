# Symmetric vs Asymmetric Cryptography

## Symmetric Encryption

Symmetric encryption uses the same key for encryption and decryption.

Advantages:
- Fast
- Efficient for large amounts of data

Disadvantages:
- Secure key sharing is difficult

Examples:
- AES
- DES

---

## Asymmetric Encryption

Asymmetric cryptography uses:

- Public key
- Private key

The public key can be shared safely while the private key remains secret.

Advantages:
- Secure key exchange
- Authentication support

Disadvantages:
- Slower than symmetric encryption

Examples:
- RSA
- ECC

---

## Diffie-Hellman

Diffie-Hellman allows secure key exchange over insecure channels.

This mechanism is widely used in secure communications.

---

## Security Importance

Modern systems combine symmetric and asymmetric encryption to balance performance and security.
