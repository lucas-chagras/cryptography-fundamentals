# Cryptography Fundamentals

This repository contains notes and practical cybersecurity concepts related to cryptography, password protection, authentication, and secure communications.

The studies were based on concepts learned through the HarvardX CS50 Introduction to Cybersecurity course and additional cybersecurity research.

---

# Topics Covered

- Hash Functions
- Password Protection
- Salting
- Symmetric Encryption
- Asymmetric Encryption
- RSA
- Diffie-Hellman
- Digital Signatures
- WebAuthn
- Encryption in Transit
- End-to-End Encryption
- Full-Disk Encryption
- Encryption at Rest
- Secure Deletion
- Quantum Computing Concepts

---

# Objectives

- Understand how modern systems protect sensitive data
- Learn cryptographic fundamentals used in cybersecurity
- Explore authentication and encryption mechanisms
- Develop foundational security knowledge for Blue Team and Security Engineering

# Key Takeaways

- Secure systems do not store plaintext passwords
- Salting significantly increases password security
- Symmetric and asymmetric cryptography solve different security problems
- Encryption protects data both in transit and at rest
- Cryptographic mechanisms are essential for authentication and confidentiality
